#  Design Patterns

##### In this folder are the laboratory exercises of class "Design Patterns". Inside of every folder is PDF with the instructions. 

---
## Lab 1

##### First laboratory exercise was about Dynamic Polymorphism and Virtual Tables. It is split into three tasks, first task is to write in C, second is to translate from C++ to C, and the third was in C++.

### Features:
   * Dynamic Polymorphism
   * Virtual, pointer tables
   * Classes, inheritance, virtual functions and static functions in C
   * Declarations of pointers

---

## Lab 2

##### Second laboratory exercise was about design principles, and the first two design patterns, strategy and observer. It was split into 6 tasks, where some of them had to be written in C, C++ and one in Python. Last few were in a language of my choice, so I chose Python. 

### Features:
   * procedural style
   * open-closed principle
   * templates
   * Statistical indicators
   * Flexible loading

---

## Lab 3

##### Third laboratory exercise was about factories that don't depend on types, so called generic factories. We had to do first task and choose one of the other three tasks that were in different languages. I chose Python.

### Features:
   * Generic Factories in C
   * Generic factories in Pyhton
   * Adding plugins
   * Dynamic libraries
